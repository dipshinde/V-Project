## Common Questions

**Q: Can I add more photos?**  
A: Yes! Edit the `gallery.items` array in config.js

**Q: What if I don't have GIFs?**  
A: Use photos instead - just change `type: "gif"` to `type: "image"`

**Q: How do I change colors?**  
A: Edit `css/style.css` - search for color codes like `#ff6b9d`

**Q: Will it work on mobile?**  
A: Yes! It's responsive. Desktop experience is better though.

**Q: Can I remove a section?**  
A: Yes, but not recommended. Every section adds to the emotional journey.
