## Option 1: GitHub Pages (Recommended)

**Cost**: Free  
**Time**: 5 minutes  
**Difficulty**: Easy  
**Custom Domain**: Yes (optional)

### Step-by-Step:

#### A. Create GitHub Account (if needed)
1. Go to [github.com](https://github.com)
2. Click "Sign up"
3. Choose a username
4. Verify email

#### B. Create Repository
1. Click the **+** icon (top right)
2. Select "New repository"
3. Repository name: `valentine-2025` (or anything you like)
4. **Keep it PUBLIC** (required for free Pages)
5. Don't check any boxes
6. Click "Create repository"

#### C. Upload Files

**Method 1: Web Interface (Easiest)**
1. Click "uploading an existing file"
2. Drag your entire `valentine-site` folder contents
3. Or click "choose your files" and select all
4. Scroll down, click "Commit changes"

#### D. Enable GitHub Pages
1. Go to repository **Settings** (top menu)
2. Click **Pages** (left sidebar)
3. Under "Source":
   - Branch: **main**
   - Folder: **/ (root)**
4. Click **Save**
5. Wait 2-5 minutes

#### E. Get Your Link
Your site will be at:
```
https://USERNAME.github.io/REPO-NAME
```
