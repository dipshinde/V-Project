## Common Last-Minute Fixes

### If photos aren't showing:
1. Check file paths in config.js
2. Ensure filenames match exactly
3. Verify files are in correct folders

### If text looks wrong:
1. Check for typos in config.js
2. Verify quotes and backticks are correct
3. Look for missing commas

### If sections aren't appearing:
1. Check browser console for errors
2. Verify config.js syntax
3. Clear cache and refresh

### If animations are broken:
1. Check all JS files are uploaded
2. Verify no console errors
3. Test in different browser
